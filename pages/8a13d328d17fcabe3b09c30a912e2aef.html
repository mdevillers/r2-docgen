Breakpoints commands

Usage: db # Breakpoints commands
| db List breakpoints
| db* List breakpoints in r commands
| db sym.main Add breakpoint into sym.main
| db Add breakpoint
| db- Remove breakpoint
| db-* Remove all the breakpoints
| db. Show breakpoint info in current offset
| dbj List breakpoints in JSON format
| dbc Run command when breakpoint is hit
| dbC Run command but continue until returns zero
| dbd Disable breakpoint
| dbe Enable breakpoint
| dbs Toggle breakpoint
| dbf Put a breakpoint into every no-return function
| dbm Add a breakpoint at an offset from a module's base
| dbn [] Show or set name for current breakpoint
| dbi List breakpoint indexes
| dbi Show breakpoint index in givengiven offset
| dbi. Show breakpoint index in current offset
| dbi- Remove breakpoint by index
| dbix [expr] Set expression for bp at given index
| dbic Run command at breakpoint index
| dbie Enable breakpoint by index
| dbid Disable breakpoint by index
| dbis Swap Nth breakpoint
| dbite Enable breakpoint Trace by index
| dbitd Disable breakpoint Trace by index
| dbits Swap Nth breakpoint trace
| dbh x86 Set/list breakpoint plugin handlers
| dbh- Remove breakpoint plugin handler
| dbt[?] Show backtrace. See dbt? for more details
| dbx [expr] Set expression for bp in current offset
| dbw Add watchpoint
| drx number addr len perm Modify hardware breakpoint
| drx-number Clear hardware breakpoint
