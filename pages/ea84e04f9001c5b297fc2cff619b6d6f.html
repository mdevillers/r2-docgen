Adding Zignatures (examples and documentation)

Zignature types:
a: bytes pattern (anal mask)
b: bytes pattern
c: base64 comment
n: real function name
g: graph metrics
o: original offset
r: references
x: cross references
h: bbhash (hashing of fcn basic blocks)
v: vars (and args)
Bytes patterns:
bytes can contain '..' (dots) to specify a binary mask

Graph metrics:
cc: cyclomatic complexity
edges: number of edges
nbbs: number of basic blocks
ebbs: number of end basic blocks

Examples:
za foo b 558bec..e8........
za foo a e811223344
za foo g cc=2 nbbs=3 edges=3 ebbs=1
za foo g nbbs=3 edges=3
za foo v b-32 b-48 b-64
za foo o 0x08048123
za foo c this is a comment (base64?)
za foo r sym.imp.strcpy sym.imp.sprintf sym.imp.strlen
za foo h 2c26b46b68ffc68ff99b453c1d30413413422d706483bfa0f98a5e886266e7ae
